# 📖 [Database From Diagram] <a name="about-project"></a>
[Database From Diagram] is a database project which uses PostGreSQL, we used a relational database to create this database and their tables from a schema diagram.

## Getting Started

This repository includes files with plain SQL that can be used to recreate a database:

- Use [schema.sql](./schema.sql) to create all tables.

### Prerequisites

In order to run this project you need:
PostGreSQL installer
Github account

### Setup

Open psql
create a new database using CREATE DATABASE [database_name]

```sh
  cd schema_based_on_diagram
  git clone https://github.com/Christelle-12/schema_based_on_diagram
```

### Install

Install this project with:
[psql installer](https://www.postgresql.org/download/)

### Usage

To run the project, execute the following command:



```sh
  \c schema_based_on_diagram
```
